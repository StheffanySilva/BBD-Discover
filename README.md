## 🚀 Tecnologias

- Ruby
- Capybara
- RSpec
- Cucumber
- Selenium WebDriver

## 👨🏻‍💻 Como executar o projeto
1. Conar o repositório, instalar as dependências
   
```
bundle install
```
2. Executar testes de regressão

      
```
bundle exec cucumber
```
